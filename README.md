## git团队协作

创建一个git裸服务器 （git init --bare）

git clone git@github.com:MLYNDHG/bookManage.git

可以进行你的开发工作了

提交：git add  你要提交的文件或者直接（git add --all）

提交：git commit -m "此次提交的备注"



推送版本至服务器 ：git remote 

推送版本至服务器 ：git push origin main

git pull从服务器下拉代码

（注：）每次提交之前一定、一定、一定要先git pull
